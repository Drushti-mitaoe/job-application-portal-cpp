# Job Application Portal (C++)

## Description
This project is a console-based Job Application Portal developed using C++ and Object-Oriented Programming principles. It simulates a real-world recruitment system where candidates can be registered, searched, ranked, and scheduled for interviews.

## Features
- Register new job applications
- Search application by ID
- Schedule and process interviews (Queue - FIFO)
- Rank candidates based on qualification score
- Update application status (Pending / Shortlisted / Rejected)
- Display all registered applications

## Concepts Used
- Object Oriented Programming (OOP)
- Linked List
- Queue
- Hash Map (map)
- Sorting techniques
- Time handling (ctime)

## How to Run
1. Compile the program:
   g++ job_application_portal.cpp -o job_app
2. Run:
   ./job_app
